# student-management-system-using-python-and-sql
---

## Sample Output
The system allows users to perform operations like adding, viewing, updating, and deleting student records. It also provides basic analysis such as average marks and top-performing students.

---

##  How to Run the Project
1. Install Python on your system
2. Clone this repository:
   git clone https://github.com/your-username/student-management-system.git

3. Navigate to the project folder:
   cd student-management-system

4. Run the Python file:
   python main.py
